Address Standardization Solution (Visual Basic .NET Edition)

Formats a Delivery Address Line according to the United States Postal
Service's Addressing Standards.

The heart of the program is in AddressStandardizationSolution.vb.

One demonstration of how to use this code is in
AddressStandardizationSolutionExample.vb.

Unit tests are in the AddressStandardizationSolutionTest directory/project.

This code is targed to the .NET Framework 2.0 in order to offer more 
flexibility in where it can be deployed.

The whole Visual Studio 2008 project is in Git to simplify future
development and collaboration.
